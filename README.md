# 🌱 Solarpunk City Planner

A 3D web-based city-building tool designed for dreaming and designing solarpunk-inspired cities. Built with Next.js, React Three Fiber, and TailwindCSS.

## 🚀 Tech Stack
- React + React Three Fiber + Drei + Zustand
- Next.js + TailwindCSS
- Hosted on Vercel
- Dev log on GitHub Pages

## 🎯 Purpose
This project serves as both a creative tool and a showcase of my technical expertise in 3D web development, systems design, and full-stack Typescript.

## 🏗️ Current Status
MVP in development. Check the [Dev Log](./src/pages/devlog.tsx) for updates.

## 🔥 How to Run
\`\`\`bash
npm install
npm run dev
\`\`\`
Then open http://localhost:3000
